 Ansible Modules:  are categorized into various groups based on their functionality.
 some of them are Listed here
   - system     ----------- Sytem modules are actions to be performed on system level, such as modifying users and groups on the sytem etc...
  
  - commands ---- command module is used to execyte commands on the hosts.
     ( command, script, shell, raw, expect)

   - files  -------  file modules are used to copy, archive, lineinfile, in a file

   - Database  ----- Database modules help in  working with databases such as MangoDB, MySql, etc... 

   - cloud  --------- The cloud section has a vast collection of clouds for various different cloud providers. like Amazon, Azure, Docker, Google.

   - windows  ----- 	windows module help in working windows environment.

  And more.....   check in "module index"
